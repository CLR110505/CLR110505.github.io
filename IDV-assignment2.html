Assignment 2 — SVG Transformations

This page is a ready-to-use template for the assignment. It shows 10 SVGs × 5 effects in a 10×5 table (original, mirrored, rotated 180°, vertically squeezed 50%, grayscale). Replace the inline SVGs below with your chosen SVGs (keep white background).

How to use
  1. Open this file and replace the SVG symbols in the svgs array (see the top of the script) with your own SVG markup. Keep each SVG as an inline <svg> element (no external file calls).
  2. Push this index.html to a GitHub repo named IDV-assignment2 and enable GitHub Pages (Settings → Pages) so your site is visible at https://<username>.github.io/IDV-assignment2.
  3. Make sure the page background stays pure white and do not pick extremely symmetric icons (e.g. plain grey circles).
